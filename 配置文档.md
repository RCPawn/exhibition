## 1. 核心架构与技术栈

## 2. 工程结构

项目采用Maven多模块聚合架构,包含以下子模块:

| 模块名称            | 端口 | 角色与功能                                                   |
| :------------------ | :--- | :----------------------------------------------------------- |
| **ruoyi-admin**     | 8080 | **主应用入口**。包含业务Controller层、启动类RuoYiApplication |
| **ruoyi-framework** | N/A  | **框架核心**。包含Spring Security配置、拦截器、数据源切换等  |
| **ruoyi-system**    | N/A  | **系统模块**。包含用户、角色、菜单、日志等基础功能           |
| **ruoyi-common**    | N/A  | **通用工具包**。包含工具类、异常定义、分页组件等             |
| **ruoyi-quartz**    | N/A  | **定时任务模块**。基于Quartz实现的任务调度                   |
| **ruoyi-generator** | N/A  | **代码生成器**。基于Velocity的CRUD代码自动生成               |
| **ruoyi-ui**        | N/A  | **前端工程**。Vue3 + Element Plus单页应用                    |

## 3. 数据库设计

| 表名                   | 说明       | 字段数 | 关键字段                                                     |
| :--------------------- | :--------- | :----- | :----------------------------------------------------------- |
